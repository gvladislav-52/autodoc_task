# Тестовое задание Autodoc
## Новостная лента iOS

Приложение, отображающее новостную ленту, полученную постранично
из [API Автодок](https://webapi.autodoc.ru/api/news/1/15).

> Проект выполнен в рамках тестового задания
> для демонстрации архитектурного подхода, качества кода
> и производительности решения.

#
## Установка
Требуется iOS 16+. Клонируйте репозиторий, откройте `.xcodeproj`
и соберите проект в Xcode. Установка пакетных менеджеров не требуется —
сторонние runtime-зависимости в проекте не используются.

#
## Описание
- Загрузка и отображение новостной ленты постранично
- Прототип ячейки содержит изображение и заголовок новости
- По нажатию на ячейку открывается новость целиком
- Адаптивная раскладка, оптимизированная для iPad

#
## Источник данных
- API: `https://webapi.autodoc.ru/api/news/1/15`
- Данные запрашиваются постранично, каждая новая страница добавляется
  в снапшот `DiffableDataSource`

#
## Технологии и архитектура
- **MVVM + Combine**
- **UICollectionView + CompositionalLayout**
- **DiffableDataSource**
- **async / await**

#
## Модули
Проект разбит на отдельные таргеты для разделения ответственности
и ускорения сборки:

| Таргет | Назначение |
| ------ | ------ |
| **ATNetworking** | Сетевой слой: клиент API, эндпоинты, модели запросов/ответов, async/await |
| **ATResources** | Ресурсы приложения: ассеты, цвета, шрифты, строки (генерируются через SwiftGen) |
| **ATUIKit** | Переиспользуемые UI-компоненты: ячейки, вью, кастомные layout'ы |
| **Основной таргет** | Экраны, вью-модели, навигация, сборка |

#
## Инструменты
Сторонние runtime-библиотеки (Alamofire, Kingfisher и т.д.) не используются,
согласно требованиям ТЗ. Применяются только инструменты для сборки:
- [SwiftGen](https://github.com/SwiftGen/SwiftGen) — типобезопасная генерация Swift-кода для ресурсов
- [SwiftLint](https://github.com/realm/SwiftLint) — контроль стиля и качества кода
